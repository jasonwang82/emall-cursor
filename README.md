# 时尚女装电商网站

一个面向中国消费者的现代化女装电商平台，设计风格参考 Zara 和 H&M。

## 项目特色

### 设计风格
- ✨ 简洁现代的用户界面
- 🎨 黑白灰主色调，搭配米色和深蓝色点缀
- 📱 完美适配移动端和桌面端
- 🎭 流畅自然的交互动效
- 🎯 使用统一风格的图标系统

### 核心功能

#### 首页
- 🖼️ 自动轮播图展示
- 🆕 新品推荐区域
- 🔥 热销商品榜单
- 👋 新用户欢迎弹窗

#### 用户系统
- 📧 邮箱注册登录
- 💬 微信登录支持
- 💳 支付宝登录支持
- 👤 用户信息管理

#### 商品展示
- 📦 商品列表页面（支持分类筛选和排序）
- 🔍 商品详情页面
  - 多图展示
  - 尺码选择
  - 用户评价展示
  - 详细的尺码对照表

#### 购物流程
- 🛒 购物车管理
- 📍 收货地址填写
- 💰 结算页面
- 💳 支付方式选择
  - 支付宝
  - 微信支付
  - 银行卡支付

#### 订单管理
- 📋 订单历史查看
- 🔍 订单状态筛选
- 📦 物流信息（准备中）

#### 客户服务
- 📧 邮件联系方式
- 📞 客服热线
- 💬 在线留言表单

## 技术栈

- **前端框架**: React 18
- **构建工具**: Vite
- **样式方案**: Tailwind CSS
- **路由管理**: React Router v6
- **图标库**: Lucide React
- **状态管理**: React Context API
- **本地存储**: LocalStorage

## 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

访问 http://localhost:3000 查看网站

### 生产构建

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```

## 项目结构

```
src/
├── components/          # 公共组件
│   ├── Header.jsx      # 页面头部
│   └── Footer.jsx      # 页面底部
├── pages/              # 页面组件
│   ├── Home.jsx        # 首页
│   ├── Login.jsx       # 登录注册
│   ├── ProductList.jsx # 商品列表
│   ├── ProductDetail.jsx # 商品详情
│   ├── Cart.jsx        # 购物车
│   ├── Checkout.jsx    # 结算页
│   ├── Orders.jsx      # 订单历史
│   └── Contact.jsx     # 联系我们
├── context/            # Context 状态管理
│   ├── CartContext.jsx # 购物车状态
│   └── AuthContext.jsx # 用户认证状态
├── App.jsx             # 应用主组件
├── main.jsx            # 应用入口
└── index.css           # 全局样式

```

## 设计规范

### 颜色主题

- **主色调**: 黑色 (#1a1a1a)
- **次要色**: 深蓝色 (#1e3a5f)
- **强调色**: 米色 (#d4c5b9)
- **中性色**: 灰度系列

### 字体

- 系统默认无衬线字体栈
- 字重: Light (300), Regular (400), Medium (500)

### 响应式断点

- 移动端: < 768px
- 平板: 768px - 1024px
- 桌面: > 1024px
