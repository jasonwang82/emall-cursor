# 时尚女装电商网站 - Fashion E-commerce

一个面向中国消费者的现代化女装电商平台，采用简约优雅的设计风格，参考 Zara 和 H&M 的设计理念。

## 技术栈

- **React 18** - 现代化前端框架
- **Vite** - 快速的构建工具
- **React Router** - 路由管理
- **TailwindCSS** - 实用优先的CSS框架
- **Lucide React** - 精美的图标库

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

服务器将在 `http://localhost:3000` 启动，并自动在浏览器中打开。

### 构建生产版本

```bash
npm run build
```

### 预览生产构建

```bash
npm run preview
```

## 核心功能说明

### 状态管理

项目使用 React Context 进行状态管理：

- **AuthContext**: 管理用户登录状态、用户信息
- **CartContext**: 管理购物车商品、数量、总价

### 数据持久化

使用 localStorage 存储：
- 用户登录信息
- 购物车数据
- 订单历史
- 首次访问标记

### 响应式设计

- 移动优先设计
- 断点：`sm: 640px`, `md: 768px`, `lg: 1024px`, `xl: 1280px`
- 移动端展示汉堡菜单
- 桌面端展示完整导航

⚠️ 本项目为演示项目，以下功能为模拟实现：

- 用户认证（无后端验证）

在生产环境中，需要：
- 集成真实的后端 API
- 实现真实的支付网关
