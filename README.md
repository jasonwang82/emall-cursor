# 女装电商网站

这是一个面向中国消费者的现代化女装电商平台，设计风格参考 Zara 和 H&M，提供简洁优雅的购物体验。

## 项目特色

- 🎨 **现代设计** - 黑白灰主色调，搭配米色和深蓝色点缀
- 📱 **响应式设计** - 完美适配移动端和桌面端
- ⚡ **快速加载** - 基于 Next.js 14 和 React 18
- 🛍️ **完整购物流程** - 从浏览商品到支付结算的全链路体验

## 核心功能

### 已实现功能

- ✅ **首页** - 轮播图、新品推荐、热销商品、新用户欢迎弹窗
- ✅ **用户认证** - 支持邮箱、微信、支付宝登录方式
- ✅ **商品浏览** - 商品列表页，支持筛选和排序
- ✅ **商品详情** - 商品图片展示、尺码选择、用户评价
- ✅ **购物车** - 添加商品、修改数量、删除商品
- ✅ **结算流程** - 地址管理、订单确认
- ✅ **支付** - 支持支付宝、微信支付、银行卡支付
- ✅ **订单管理** - 查看历史订单、订单状态跟踪

### 设计亮点

- 简洁的导航栏，固定顶部，带搜索功能
- 流畅的页面过渡和交互动效
- 统一的图标风格（使用 Lucide React）
- 清晰的视觉层级和信息架构
- 完善的无障碍体验

## 技术栈

- **框架**: Next.js 14 (App Router)
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **图标**: Lucide React
- **图片**: Next.js Image 优化

## 开始使用

### 安装依赖

```bash
npm install
```

### 运行开发服务器

```bash
npm run dev
```

访问 [http://localhost:3000](http://localhost:3000) 查看网站。

### 构建生产版本

```bash
npm run build
npm start
```

## 项目结构

```
├── app/                    # Next.js App Router 页面
│   ├── page.tsx           # 首页
│   ├── login/             # 登录注册页
│   ├── products/          # 商品列表和详情页
│   ├── cart/              # 购物车页
│   ├── checkout/          # 结算页
│   ├── payment/           # 支付页
│   └── orders/            # 订单页
├── components/            # 可复用组件
│   ├── Header.tsx        # 导航栏
│   ├── Footer.tsx        # 页脚
│   ├── Modal.tsx         # 模态框
│   ├── Carousel.tsx      # 轮播图
│   ├── ProductCard.tsx   # 商品卡片
│   └── ...
├── public/               # 静态资源
└── styles/              # 全局样式
```

## 页面展示

- **首页** - `/` - 展示轮播图、新品和热销商品
- **商品列表** - `/products` - 浏览所有商品
- **商品详情** - `/products/[id]` - 查看商品详细信息
- **登录注册** - `/login` - 用户认证
- **购物车** - `/cart` - 管理购物车商品
- **结算** - `/checkout` - 填写收货地址
- **支付** - `/payment` - 选择支付方式
- **订单** - `/orders` - 查看历史订单
