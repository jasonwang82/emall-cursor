# 时尚女装电商平台 - FASHION

一个现代化、简洁的女装电商网站，专为中国消费者打造，设计风格参考 Zara 和 H&M。

## 项目特点

- ✨ **现代简洁**：黑白灰主色调，搭配米色和深蓝色点缀
- 📱 **移动优先**：完全响应式设计，完美适配各种设备
- 🎨 **优雅交互**：流畅的动画效果和直观的用户体验
- 🛒 **完整流程**：从浏览商品到支付的完整购物体验
- 🔒 **安全支付**：支持支付宝、微信支付、银行卡等主流支付方式

### 🏠 首页

- 邮件联系

- 在线客服（开发中）

## 技术栈

- **前端框架**：React 18 + TypeScript
- **构建工具**：Vite
- **路由**：React Router v6
- **样式**：Tailwind CSS
- **图标**：Lucide React

## 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

访问 http://localhost:3000 查看网站

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```

## 设计规范

### 色彩方案
- **主色调**：黑白灰
- **点缀色**：米色 (#f5f1e8)、深蓝色 (#2c3e50)
- **强调色**：纯黑 (#212529)

### 字体
- 系统默认字体栈，优先使用 PingFang SC、Hiragino Sans GB、Microsoft YaHei

### 间距
- 遵循 Tailwind 的间距系统
- 统一使用 4px 基准间距
