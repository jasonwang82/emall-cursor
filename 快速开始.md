# 快速开始指南

## 📦 安装步骤

1. **安装依赖**
```bash
npm install
```

2. **启动开发服务器**
```bash
npm run dev
```

项目将在浏览器自动打开 `http://localhost:3000`

## 🎯 功能演示路径

### 1️⃣ 首页体验
- 访问首页查看轮播图、新品推荐和热销商品
- 首次访问会看到新用户欢迎弹窗

### 2️⃣ 浏览商品
- 点击导航栏的分类（连衣裙、上装、下装、外套）
- 使用筛选器按价格和分类过滤商品
- 点击商品查看详情页

### 3️⃣ 购物流程
1. 在商品详情页选择尺码和数量
2. 点击"加入购物车"或"立即购买"
3. 查看购物车，调整商品数量
4. 点击"去结算"进入结算页面
5. 填写收货地址信息
6. 选择支付方式（支付宝/微信/银行卡）
7. 确认支付完成订单

### 4️⃣ 用户功能
- **注册/登录**: 支持邮箱、微信、支付宝三种方式
- **我的订单**: 登录后点击用户图标 → 我的订单
- **联系我们**: 底部导航 → 联系我们

## 🎨 设计特色

### 色彩方案
- **主色调**: 黑色 (#1a1a1a)、白色 (#ffffff)、灰色 (#6b6b6b)
- **辅助色**: 米色 (#f5f1ed)、深蓝色 (#2c3e50)

### 响应式断点
- **移动端**: < 768px - 汉堡菜单、单列商品
- **平板**: 768px - 1024px - 2列商品网格
- **桌面**: > 1024px - 完整导航、多列布局

### 交互动效
- 图片悬停放大效果
- 平滑的页面过渡
- 自然的按钮动画
- 流畅的轮播切换

## 📱 移动端优化

- 触摸友好的按钮大小
- 优化的图片加载
- 移动端导航菜单
- 快速响应的交互

## 🔧 技术细节

### 状态管理
```javascript
// 购物车状态
const { cartItems, addToCart, removeFromCart } = useCart()

// 用户认证状态
const { user, login, logout, isAuthenticated } = useAuth()
```

### 数据存储
- localStorage 用于持久化：
  - 用户信息
  - 购物车数据
  - 订单历史
  - 首次访问标记

## 🚀 构建部署

## 💡 使用提示

1. **模拟数据**: 当前使用的是前端模拟数据，实际项目需要对接后端API
2. **支付功能**: 支付流程为演示性质，未集成真实支付网关
3. **图片资源**: 使用 Unsplash 的示例图片
4. **用户认证**: 登录功能为模拟实现，无后端验证
