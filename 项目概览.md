# 时尚女装电商网站 - 项目概览

## 📋 项目信息

**项目名称**: Fashion E-commerce - 时尚女装电商平台  
**设计风格**: 参考 Zara 和 H&M 的简约现代风格  
**目标用户**: 中国消费者  
**设备支持**: 移动端优先，全响应式设计

## 🎨 设计实现

### 色彩系统
```javascript
colors: {
  primary: {
    black: '#1a1a1a',      // 主黑色
    gray: '#6b6b6b',       // 灰色
    lightgray: '#e5e5e5',  // 浅灰色
    white: '#ffffff',      // 白色
    beige: '#f5f1ed',      // 米色
    navy: '#2c3e50'        // 深蓝色
  }
}
```

### 组件样式类
- ✅ `.btn-primary` - 主要按钮（黑底白字）
- ✅ `.btn-secondary` - 次要按钮（白底黑字）
- ✅ `.input-field` - 输入框统一样式
- ✅ `.section-title` - 区块标题
- ✅ `.card-hover` - 卡片悬停效果

### 响应式断点
- ✅ `sm: 640px` - 小屏幕
- ✅ `md: 768px` - 中等屏幕
- ✅ `lg: 1024px` - 大屏幕
- ✅ `xl: 1280px` - 超大屏幕

### 交互动效
- ✅ 图片悬停缩放（scale-105）
- ✅ 按钮悬停颜色变化
- ✅ 平滑的过渡效果（transition-all duration-300）
- ✅ 轮播图淡入淡出
- ✅ 弹窗动画
- ✅ 自定义滚动条样式

## 📦 技术栈

- **React 18.2.0** - UI框架
- **React Router 6.20.0** - 路由管理
- **Vite 5.0.8** - 构建工具
- **TailwindCSS 3.3.6** - CSS框架
- **Lucide React 0.294.0** - 图标库
- **PostCSS** - CSS处理
- **Autoprefixer** - CSS自动前缀
