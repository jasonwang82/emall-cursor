# 快速启动指南

## 安装步骤

### 1. 安装依赖

```bash
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

### 3. 访问网站

在浏览器中打开 [http://localhost:3000](http://localhost:3000)

## 主要页面路由

| 页面 | 路径 | 说明 |
|------|------|------|
| 首页 | `/` | 轮播图、新品、热销商品 |
| 商品列表 | `/products` | 所有商品展示，支持筛选 |
| 商品详情 | `/products/1` | 商品详细信息 |
| 登录注册 | `/login` | 用户认证 |
| 购物车 | `/cart` | 购物车管理 |
| 结算 | `/checkout` | 填写收货地址 |
| 支付 | `/payment` | 支付方式选择 |
| 订单 | `/orders` | 历史订单查看 |

## 项目特点

### 设计风格
- **配色方案**: 黑白灰主色调 + 米色/深蓝色点缀
- **排版**: 简洁现代，大量留白
- **图片**: 高质量商品图，3:4 比例
- **字体**: 使用系统字体，支持中文优化

### 响应式设计
- **移动端优先**: 完美适配手机浏览
- **断点**: sm (640px), md (768px), lg (1024px)
- **交互**: 触摸友好的按钮和表单

### 核心功能

#### 1. 首页亮点
- 自动轮播的 Hero 图（5秒切换）
- 新品推荐区域
- 热销商品展示
- 新用户欢迎弹窗（仅首次访问显示）

#### 2. 商品浏览
- 分类筛选（上衣、连衣裙、裤装等）
- 多维度筛选（尺码、颜色、价格）
- 排序功能（价格、最新）
- 加载更多功能

#### 3. 商品详情
- 多图展示（4张）
- 尺码选择器
- 颜色选择器
- 数量调节
- 用户评价展示

#### 4. 购物流程
- 购物车实时更新
- 运费计算（满299免运费）
- 地址管理（新增、选择）
- 多种支付方式

#### 5. 用户系统
- 邮箱登录
- 微信登录（显示二维码）
- 支付宝登录（显示二维码）
- 订单历史追踪

## 技术细节

### 状态管理
目前使用 React 本地状态（useState）
后续可集成：
- Zustand (轻量级状态管理)
- React Context (全局状态)

### 数据获取
使用模拟数据（Mock Data）
生产环境可替换为：
- REST API
- GraphQL
- Next.js API Routes

### 图片优化
- 使用 Next.js Image 组件
- 自动格式转换（WebP）
- 响应式图片加载
- 懒加载支持

### 样式系统
- Tailwind CSS 实用类
- 自定义动画（fadeIn, slideDown 等）
- 响应式工具类
- 深色模式支持（可扩展）

## 自定义配置

### 修改颜色主题

编辑 `tailwind.config.js`:

```js
colors: {
  primary: {
    DEFAULT: '#1a1a1a',  // 修改主色
    light: '#2d2d2d',
  },
  // ...
}
```

### 修改轮播图

编辑 `components/Carousel.tsx` 中的 `slides` 数组

### 添加商品分类

编辑 `components/Header.tsx` 中的导航链接

## 常见问题

**Q: 图片加载失败？**
A: 确保网络可访问 Unsplash 或替换为本地图片

**Q: 如何添加真实数据？**
A: 替换各页面中的 mock 数据，连接后端 API

**Q: 支持服务端渲染吗？**
A: 是的，使用 Next.js App Router，支持 SSR 和 SSG

**Q: 如何修改网站名称？**
A: 修改 `components/Header.tsx` 中的 "FASHION" 文本
